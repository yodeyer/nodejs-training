Présentation

Node.js est un environnement d'éxécution JavaScript. Il est construit sur le moteur JavaScript V8 de Chrome et se distingue des autres plateformes par son approche non bloquante permettant d'effectuer des entrées / sorties (I/O) de façon asynchrone.


  • libuv est une bibliothèque C multi-plateforme qui prend en charge les entrées / sorties asynchrones basées sur des boucles d'événement
  • V8 est un moteur JavaScript libre et open source développé par Google en C++

JavaScript est un langage single-threaded : le code s'exécute sur un seul et unique thread; aucun appel n'est dispatché. Node.js se base également sur le même principe, il est architecturé sur le modèle : "Single Thread Event Loop Model"

Historique

Quelques dates importantes :

  • 1995 : Création du langage LiveScript, inspiré du Java, interprété conçu pour écrire des serveur HTTP. Il sera renommé en JavaScript dans sa version intégré par Netscape;
  • 1997 : Spécification du langage par l'ECMA;
  • 2006 : Lancement de JQuery;
  • 2008 : Le moteur V8 est intégré dans Chrome et Chromium;
  • 2009 : Création de Node.js;
  • 2015 : EcmaScript 2015 (ES6) est approuvé le 17 juin;
  • 2018 : Node.js V10 en LTS (Support à Long Terme) le 24 avril.
